base url:
    GET
    https://uapis.cn/api/v1/social/bilibili/archives?mid={{UID}}&ps=50&pn={{Page}}

这个api适用于获取用户发布到视频详情，包含bvid和标题。播放数量等字段
在这个项目中，网页调用时应该获取全部的视频bvid、标题

参数说明
mid (用户ID): B站用户的mid，必填参数。
keywords (搜索关键词): 可选，用于在该UP主的投稿中搜索特定关键词。
orderby (排序方式):
pubdate: 按最新发布排序（默认）
views: 按最多播放排序
ps (每页条数): 默认20条。
pn (页码): 默认1，用于分页。
响应体字段说明
total (总稿件数): UP主的投稿总数。
page (当前页码): 当前返回的页码。
size (每页数量): 每页返回的视频数量。
videos (视频列表): 包含当前页的所有视频信息：
aid: 视频的AV号
bvid: 视频的BV号
title: 视频标题
cover: 封面URL
duration: 时长（秒）
play_count: 播放量
publish_time: 发布时间戳
create_time: 创建时间戳
state: 视频状态
is_ugc_pay: 是否付费视频（0=免费，1=付费）
is_interactive: 是否为互动视频

1.无视频
```json
{
  "total": 0,
  "page": 1,
  "size": 50,
  "videos": []
}
```
